How to play this game?

❌

The objectives of this game is to guess the number randomly generated by the app.
User have to insert a number and click on the GUESS BUTTON to play, a message is displayed.
User start with a total of 30 points and on every wrong guess the user lose 5 points.
If you guess the number before the number of total attempts ends the current score is saved as the highest score
So, you have to beat the previous Highscore if you click RESTART BUTTON.
